# 高校选课信息系统

一个简单易用的高校选课系统，基于 Flask + SQLite 开发。

## 功能特性

- 📚 课程浏览：查看所有可选课程信息
- ✅ 在线选课：学生可以选择心仪的课程
- ❌ 退课功能：支持退选已选课程
- 📊 容量管理：实时显示课程人数和容量
- 🎯 简单易用：界面简洁，操作方便

## 安装步骤

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 初始化数据库

运行初始化脚本，创建示例数据：

```bash
python init_data.py
```

这会创建3个示例学生和5门示例课程：

**学生信息：**
- 学号：2021001，姓名：张三，专业：计算机科学
- 学号：2021002，姓名：李四，专业：软件工程
- 学号：2021003，姓名：王五，专业：数据科学

**课程信息：**
- CS101 - 数据结构（陈教授）
- CS102 - 算法设计（刘老师）
- CS103 - 操作系统（王教授）
- CS104 - 数据库原理（张老师）
- CS105 - 计算机网络（李教授）

### 3. 启动系统

```bash
python app.py
```

### 4. 访问系统

打开浏览器访问：http://127.0.0.1:5000

## 使用说明

1. **输入学号**：在页面顶部输入学号（如：2021001）
2. **浏览课程**：查看"可选课程"列表，绿色状态表示可选，红色表示已满
3. **选择课程**：点击"选课"按钮进行选课
4. **查看已选课程**：在"我的课程"区域查看已选课程
5. **退选课程**：点击"退课"按钮退选课程

## 项目结构

```
.
├── app.py              # 主应用程序（包含数据库模型和API）
├── init_data.py        # 数据初始化脚本
├── requirements.txt    # Python依赖包
├── templates/
│   └── index.html     # 前端页面
└── course.db          # SQLite数据库（运行后自动生成）
```

## API接口

- `GET /api/courses` - 获取所有课程列表
- `GET /api/my-courses/<student_id>` - 获取学生已选课程
- `POST /api/enroll` - 选课
- `POST /api/drop` - 退课

## 技术栈

- **后端**：Flask 3.0、Flask-SQLAlchemy 3.1
- **数据库**：SQLite
- **前端**：HTML、CSS、JavaScript

## 注意事项

- 学生选课前需要先输入学号
- 课程满员后将无法继续选课
- 每个学生不能重复选择同一门课程
- 退课后会释放课程名额

## 开发说明

如需添加更多学生或课程，可以修改 `init_data.py` 文件，然后重新运行初始化脚本。
